# mugenseichi
# Modpack for Beginners of Flattening / 整地初心者向けModpack

## 特徴 / Features
整地初心者のためのModpackです。  
一括破壊と無線MEターミナルを使用して、効率的な整地ができます。

----
This is a modpack designed for beginners who want to start terrain flattening.  
You can efficiently flatten terrain using vein mining and a wireless ME terminal.

---

## 環境 / Requirements
- Minecraft 1.20.1  
- Forge 47.2.0
  
---

## 導入方法 / Installation
CurseForge Launcher などのアプリを使用してください。  

1. Zipファイルをインポート  
2. プロファイルを作成  
3. Minecraftを起動  

----
Please use an application such as the CurseForge Launcher.

1. Import the zip file  
2. Create a profile  
3. Launch Minecraft
